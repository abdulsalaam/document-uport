# document-uport

#### Contributing

###### Getting started
```
$ git clone git@github.com:uport-project/demo.git
$ yarn install
```

###### Development
```
npm run start
```

###### Deployment
*You will need Amazon S3 permissions to deploy - Contact [andres](https://www.google.com)*

To deploy to **demo.uport.me**
```
npm run deploy
```
To deploy to **demo.uport.space**
```
npm run deploy-test
```
